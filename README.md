# Compile Environment

## Requirement

### iOS Requirement
- xCode 10.0
- Cocoapods


### Android Requirement
- Android Studio 3.0.1


### Common Requirement
- Node Package Manager
- React Native CLI

## To Compile

1. Clone this repo.
2. Go to root folder in terminal and use npm install
3. For iOS Go to iOS Folder in terminal and use pod install to install pod
4. For android open android studio.
